# SequelProCopyAsMarkdown

Copy selected rows as a Markdown. This supports Japanese.

## Installation
Installing a bundle is as simple as downloading an .spbundle file from my repository and double-clicking on it from your Downloads folder.

## Usage
Start by connecting to a database, and choosing a selection of data from one of your tables. Right click on the selected data and choose Bundles> CopyAsMarkdown. This will copy the Markdown formatted code to your Pasteboard.
